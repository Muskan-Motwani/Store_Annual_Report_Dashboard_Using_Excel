# 📊 Interactive Sales Dashboard in Microsoft Excel

## 📌 Project Overview

This project demonstrates how raw sales data can be transformed into meaningful business insights using **Microsoft Excel**. The dataset was cleaned, processed, analyzed, and visualized through an interactive dashboard that enables users to explore sales performance across different dimensions using slicers and dynamic charts.

The objective of this project is to showcase Excel's powerful data analysis capabilities using Pivot Tables, Pivot Charts, and Dashboard design techniques.

## 📈 Dashboard Highlights

The dashboard provides interactive analysis of sales data through:

* Sales Performance Overview
* Sales by Category
* Sales by Region
* Sales by Customer Segment
* Sales Trends
* Dynamic Filtering using Slicers

Users can filter the dashboard to instantly update all charts and visualizations.
